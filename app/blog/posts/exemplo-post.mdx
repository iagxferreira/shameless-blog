# Como usar o React useState

O hook `useState` é uma das formas mais simples de adicionar estado a componentes funcionais no React. Veja um exemplo básico de uso:

```jsx
import React, { useState } from "react";

function Contador() {
  const [contador, setContador] = useState(0);

  return (
    <div>
      <p>Você clicou {contador} vezes</p>
      <button onClick={() => setContador(contador + 1)}>Clique aqui</button>
    </div>
  );
}
```

A cada clique no botão, o valor do contador é incrementado.
